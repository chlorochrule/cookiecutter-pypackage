# cookiecutter pypackage for chlorochrule

## Installation

Install cookiecutter.

```bash
pip install cookiecutter
```

## How to use

Create a new package.

```bash
cookiecutter https://github.com/chlorochrule/cookiecutter-pypackage.git
```
